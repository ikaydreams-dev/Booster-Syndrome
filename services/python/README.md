# Python Services

Python-based services for ML, data processing, and web scraping.

## Services

- **machine_learning.py** - ML algorithms (Linear/Logistic Regression, KMeans, Decision Trees, Neural Networks)
- **image_processing.py** - Image manipulation and filters
- **web_scraping.py** - Web crawler and HTML parser
- **async_patterns.py** - Async utilities (pools, caching, event emitters)
- **data_structures.py** - Advanced data structures (Trie, Segment Tree, Skip List)

## Setup

```bash
pip install -r requirements.txt
pytest
```
