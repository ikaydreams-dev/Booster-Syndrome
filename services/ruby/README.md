# Ruby Services

This directory contains all Ruby-based microservices.

## Services

- **authentication.rb** - OAuth2, JWT, 2FA authentication
- **web_framework.rb** - Web framework with routing and middleware
- **orm_layer.rb** - ORM with migrations and query builder
- **testing_framework.rb** - Test framework with mocks and spies
- **actor_model.rb** - Actor model with GenServer
- **reactive_streams.rb** - Reactive programming with observables
- **functional_core.rb** - Functional programming monads
- **service_layer.rb** - Service objects and patterns
- **dsl_framework.rb** - DSL builders
- **pattern_matching.rb** - Pattern matching implementation
- **metaprogramming.rb** - Metaprogramming utilities
- **type_system.rb** - Type checking and contracts

## Setup

```bash
bundle install
bundle exec rspec
```
