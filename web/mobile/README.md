# Mobile Application

React Native mobile app for iOS and Android.

## Features

- Cross-platform (iOS/Android)
- React Navigation
- Redux state management
- Native performance
- Push notifications

## Tech Stack

- React Native
- TypeScript
- Redux Toolkit
- React Navigation
- Axios

## Running

```bash
# Install dependencies
npm install

# iOS
npm run ios

# Android
npm run android

# Metro bundler
npm start
```

## Building

```bash
# iOS
cd ios && pod install
npm run ios

# Android
npm run android
```

## Features

- User authentication
- Real-time chat
- Push notifications
- Offline support
- File uploads
