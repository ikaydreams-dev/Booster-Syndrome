# Admin Dashboard

React TypeScript admin dashboard for system management.

## Features

- User management
- System monitoring
- Analytics dashboard
- Service configuration
- Audit logging

## Tech Stack

- React 18
- TypeScript
- Redux Toolkit
- TailwindCSS
- Chart.js for charts
- React Table

## Running

```bash
# Install dependencies
npm install

# Development
npm run dev

# Build
npm run build
```

## Access Control

Admin dashboard requires admin role and valid JWT token.

## Features

- User CRUD operations
- Real-time system metrics
- Service health monitoring
